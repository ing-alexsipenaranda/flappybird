# 📸 Instrucciones para las 3 Imágenes del README

## Imágenes Requeridas

Necesitas colocar **3 imágenes** en esta carpeta con estos nombres exactos:

### 1. `screenshot1.png` - Pantalla de Inicio
- Captura la pantalla inicial del juego
- Debe mostrar el título, selector de dificultad y botones

### 2. `screenshot2.png` - Gameplay
- Captura el juego en acción
- Debe mostrar el pájaro, tuberías y puntuación

### 3. `screenshot3.png` - Game Over
- Captura la pantalla de Game Over
- Debe mostrar la puntuación final y botones de reinicio

## 🚀 Cómo Tomar las Capturas

1. **Inicia el servidor**:
   ```bash
   cd /Users/ingalexsi/FLAPPYBIRD/flappy-bird
   python3 -m http.server 8080
   ```

2. **Abre el juego**: http://localhost:8080

3. **Toma las capturas**:
   - **Mac**: `Cmd + Shift + 4` (selecciona el área)
   - **Windows**: `Win + Shift + S`

4. **Guarda las imágenes** en esta carpeta con los nombres exactos:
   - `screenshot1.png`
   - `screenshot2.png`
   - `screenshot3.png`

## ✅ Verificación

Después de agregar las imágenes, verifica:

```bash
ls -lh screenshots/screenshot*.png
```

Deberías ver:
```

screenshot2.png
screenshot3.png
```
